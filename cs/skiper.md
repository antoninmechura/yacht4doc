# Průvodce pro skipera — internet a lodní data

---

## 1. Internet na lodi

Router je připojen přes mobilní síť. Pokud chcete používat internet pro sebe a posádku, **vložte svou SIM kartu do slotu SIM1** v routeru RUTX11.

> Router se nachází v _______________________ *(doplní majitel)*.

**Postup:**
1. Otevřete kryt SIM slotů na routeru
2. Vložte SIM kartu do slotu označeného **SIM 1**
3. Počkejte přibližně **2 minuty** — router automaticky přepne na vaši SIM
4. Připojte se k WiFi síti a internet bude fungovat přes vaši SIM

**WiFi síť:**
- Název (SSID): `RUT_SOUL_2G`
- Heslo: __________________ *(doplní majitel)*

> Po vyplutí nebo přistání jednoduše SIM vytáhněte — router se vrátí na záložní síť majitele.

## 2. Lodní data — web dashboard

Po připojení k WiFi si v prohlížeči otevřete:

**`http://192.168.100.3`**

Dashboard se automaticky obnovuje každé 2 sekundy a zobrazuje:

| Sekce | Data |
|-------|------|
| Navigace | Poloha, SOG, COG, kurz, STW, log |
| Vítr | Zdánlivý vítr (AWS/AWA), pravý vítr (TWS/TWD) |
| Hloubka | Hloubka pod kýlem |
| Motory | RPM, teplota chladicí kapaliny, tlak oleje, palivo, napětí, motohodiny |
| Prostředí | Teplota moře, teplota vzduchu, barometrický tlak |
| Náklon | Heel (příčný náklon), trim (podélný náklon) |
| Baterie | Napětí, proud, SoC (pokud dostupné) |

> Data jsou zobrazována pouze pokud jsou k dispozici — prázdné pole nebo `---` znamená, že přístroj momentálně neposílá data.
